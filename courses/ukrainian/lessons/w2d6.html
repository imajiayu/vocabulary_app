1. 语法讲解：不规则复数名词

大部分乌克兰语名词都遵循上面的规则变复数，但有少数常用名词的复数形式是不规则的，需要单独记忆。今天我们按类型逐一学习。

A. 完全不规则（词干大变）

这些名词的复数形式与单数差别很大，只能死记硬背：

单数中文复数变化说明
дитина孩子діти完全变干，词根大变
людина人люди完全变干，词根大变
ім'я名字імена加 -ена
око眼睛очі词干变化 к→ч，词尾变 і
вухо耳朵вуха看似规则（-о→-а），但归为特殊记忆
记忆技巧：
дитина（孩子）→ діти（孩子们）—— 想象一群 di-ti（弟弟）在玩耍
людина（人）→ люди（人们）—— 很像英文 "people"，单复数也是不同词
око（眼睛）→ очі（眼睛们）—— к 变成了 ч，就像眼睛「眨」了一下变了样
注意：дитина 和 людина 是第1周已学的词汇，但当时我们只学了它们的单数形式和性别（都是阴性 -а 结尾）。今天我们补充它们的不规则复数。ім'я 也是第1周学过的（中性特殊词 -м'я 类）。

B. 幼崽名词（-а/-я → -ата/-ята）

这是乌克兰语的一大特色！所有动物幼崽的名词，单数以 -а 或 -я 结尾，复数统一变为 -ата 或 -ята。

变化规则：

  • 单数以 -а 结尾 → 复数去掉 -а，加 -ата
  • 单数以 -я 结尾 → 复数去掉 -я，加 -ята
单数中文复数规律
курча小鸡курчата-а → -ата
лоша小马驹лошата-а → -ата
теля小牛телята-я → -ята
щеня小狗щенята-я → -ята
кошеня小猫кошенята-я → -ята
козеня小山羊козенята-я → -ята
порося小猪поросята-я → -ята
каченя小鸭каченята-я → -ята
маля宝宝/幼仔малята-я → -ята
记忆技巧：看到动物幼崽，就想到 -ата/-ята！
可以这样记：幼崽们总是成群结队（-ата/-ята 听起来就像一群小动物叽叽喳喳的声音）。

快速判断法：如果一个词指的是「小XX」（小鸡、小猫、小狗……），它的复数几乎一定是 -ата/-ята。
易错提醒：
幼崽名词虽然单数以 -а/-я 结尾（看起来像阴性），但它们实际上是中性名词！
例如：теля（小牛）是中性，不是阴性。
代词用 воно，不用 вона！
记住：幼崽名词 = 中性 + 不规则复数 -ата/-ята

D. 少数无复数或复数罕用的名词

有些名词通常只以单数形式使用，没有复数或复数极少见：

  • молоко（牛奶）— 不可数，没有复数
  • пиво（啤酒）— 可说 пива（复数），但罕用
  • волосся（头发）— 集合名词，本身就表示复数概念
  • листя（树叶）— 集合名词，本身就表示复数概念

E. 其他需要注意的复数形式

单数中文复数说明
перо羽毛пера按中性 -о→-а 规则（也有 пір'я 的集合形式）
плече肩膀плечі-е → -і（特殊变化）
коліно膝盖коліна按中性 -о→-а 规则
-м'я 类词的复数模式：
ім'я → імена
规律：去掉 -'я，加 -ена。

2. 例句

以下例句展示不规则复数名词的用法。使用 Це（这是）、тут（在这里）、там（在那里）句式。

完全不规则复数

1. Це діти. — 这些是孩子们。（дитина → діти）

2. Люди тут. — 人们在这里。（людина → люди）

3. Це імена. — 这些是名字。（ім'я → імена）

4. Очі тут. — 眼睛在这里。（око → очі）

5. Це вуха. — 这些是耳朵。（вухо → вуха）

幼崽名词复数

6. Курчата тут. — 小鸡们在这里。（курча → курчата）

7. Це телята. — 这些是小牛。（теля → телята）

8. Щенята там. — 小狗们在那里。（щеня → щенята）

9. Це кошенята. — 这些是小猫。（кошеня → кошенята）

10. Козенята тут. — 小山羊们在这里。（козеня → козенята）

11. Поросята там. — 小猪们在那里。（порося → поросята）

12. Це лошата. — 这些是小马驹。（лоша → лошата）

13. Каченята тут. — 小鸭们在这里。（каченя → каченята）

14. Малята там. — 宝宝们在那里。（маля → малята）

其他不规则及身体部位

15. Це пера. — 这些是羽毛。（перо → пера）

16. Плечі тут. — 肩膀在这里。（плече → плечі）

17. Це коліна. — 这些是膝盖。（коліно → коліна）

单复数对比

19. Око тут, очі там. — 一只眼睛在这里，两只眼睛在那里。

20. Дитина тут, діти там. — 孩子在这里，孩子们在那里。

21. Щеня тут, щенята там. — 小狗在这里，小狗们在那里。

4. 今日小结

今日要点回顾

今天我们学习了乌克兰语中的不规则复数名词，主要包括以下几类：

  1. 完全不规则：дитина→діти，людина→люди，око→очі — 只能死记硬背
  2. -м'я 类：ім'я→імена — 去掉 -'я，加 -ена
  3. 幼崽名词：-а→-ата / -я→-ята — 这是最重要的规律！
    курча→курчата，теля→телята，щеня→щенята……
  4. 其他特殊：плече→плечі，вухо→вуха
