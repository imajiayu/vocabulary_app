学习目标：掌握中性名词的复数变化规则（-о→-а、-е→-я、-ння/-ття 同形、外来词不变），学会15个新中性名词，并能综合运用三种性别的复数规则。

1. 回顾：阳性与阴性名词复数

前两节课要点回顾

性别词尾复数规则示例
阳性硬辅音加 -ибрат → брати
阳性软辅音/嘶嘶音加 -іолівець → олівці

2. 语法讲解：中性名词复数规则

今天是名词复数系列的最后一课！中性名词的复数变化与阳性、阴性有明显不同。中性名词复数的标志性词尾是 -а 和 -я（而不是 -и/-і）。

有趣的对比：阴性名词以 -а/-я 结尾，复数变成 -и/-і；中性名词以 -о/-е 结尾，复数反而变成 -а/-я。可以说，中性名词的复数词尾看起来像阴性名词的单数词尾！不要搞混了。

规则1：-о 结尾 → 去掉 -о，加 -а

这是最常见的中性复数规则。以 -о 结尾的中性名词，复数时把 -о 换成 -а。

单数复数中文
вікновікна窗户
словослова单词
озероозера湖
селосела村庄
дзеркалодзеркала镜子

记忆技巧：中性名词 -о → -а，就是把最后一个字母"倒过来"：о 是圆的，а 是尖的。圆变尖！

注意不可数名词！

有些以 -о 结尾的中性名词是不可数名词，没有复数形式：

  • молоко（牛奶）— 不可数，无复数
  • масло（黄油/油）— 通常不可数

这和中文/英文一样，"牛奶""黄油"本身就不说"很多个牛奶"。

规则2：-е 结尾 → 去掉 -е，加 -я

以 -е 结尾的中性名词，复数时把 -е 换成 -я。

单数复数中文
мореморя海
полеполя田野
серцесерця心脏
місцемісця地方
сонцесонця太阳

记忆技巧：-е → -я，和规则1是配对的：硬 -о 配硬 -а，软 -е 配软 -я。还是硬配硬，软配软的规律！

中性复数的核心规律：
  • 硬元音 -о → 硬元音 -а：вікно → вікна
  • 软元音 -е → 软元音 -я：море → моря

这和之前学的"硬配-и、软配-і"是同一个大规律的延伸。乌克兰语里，硬软配对无处不在！

规则3：-ння / -ття 结尾 → 复数形式与单数相同！

以 -ння 或 -ття 结尾的中性名词，复数形式和单数完全一样，需要靠语境来区分。

单数复数中文
завданнязавдання作业/任务
питанняпитання问题
знаннязнання知识
життяжиття生命
почуттяпочуття感觉

记忆技巧：-ння/-ття 结尾的词"太长了，懒得变"。单复数同形，靠上下文判断。这类词在中文里也类似 — "一个问题"和"很多问题"，"问题"这个词本身没变。

规则4：不变化的外来词 → 复数也不变！

我们在第1周学过的不变化外来词（中性），它们的复数和单数完全一样。

单数复数中文
кафекафе咖啡馆
метрометро地铁
таксітаксі出租车
кінокіно电影
менюменю菜单
радіорадіо广播

记忆技巧：外来词"入乡不随俗"，来到乌克兰语也不肯变化。单数复数都一样，全靠语境。

中性复数规则总结

词尾变化方式示例

-е去 -е 加 -яморе → моря
-ння不变（同形）завдання → завдання
-ття不变（同形）життя → життя

易混淆警告：中性复数 -а 和阴性单数 -а

中性名词的复数词尾是 -а，这和阴性名词的单数词尾一样！不要搞混：

  • вікна — 是 вікно（窗户）的复数，不是阴性单数
  • школа — 是阴性名词的单数

区分方法：看这个词你是否认识。如果认识它的单数形式（如 вікно），就知道 вікна 是它的复数。

5. 今日小结

中性名词复数 — 四大规则

  1. -о 结尾 → 去-о 加-а：вікно→вікна，крісло→крісла，колесо→колеса
  2. -е 结尾 → 去-е 加-я：море→моря，серце→серця，місце→місця
  3. -ння/-ття 结尾 → 不变：завдання→завдання，життя→життя
  4. 外来词 → 不变：кафе→кафе，пальто→пальто

三种性别复数的总口诀：

  • 阳性：硬辅+и，软辅+і
  • 阴性：-а→-и，-я→-і，-ь→-і
  • 中性：-о→-а，-е→-я，-ння/-ття/外来词不变

核心规律：硬配硬（-о↔-а，辅音+и），软配软（-е↔-я，软辅+і）。
