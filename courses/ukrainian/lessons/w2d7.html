2. 综合例句

以下例句覆盖单数和复数，混合三种性别。使用 Це（这是）、тут（在这里）、там（在那里）句式。

1. Це столи. — 这些是桌子们。（阳性：стіл→столи）

2. Тарілки тут. — 盘子们在这里。（阴性：тарілка→тарілки）

3. Це крісла. — 这些是扶手椅。（中性：крісло→крісла）

4. Друзі там. — 朋友们在那里。（阳性不规则：друг→друзі，г→з）

5. Це сукні. — 这些是裙子。（阴性：сукня→сукні，-я→-і）

6. Діти тут. — 孩子们在这里。（完全不规则：дитина→діти）

7. Це ножі. — 这些是刀。（阳性：ніж→ножі，飘忽元音+ж→жі）

8. Кімнати там. — 房间们在那里。（阴性：кімната→кімнати，-а→-и）

9. Це колеса. — 这些是轮子。（中性：колесо→колеса，-о→-а）

10. Щенята тут. — 小狗们在这里。（幼崽：щеня→щенята，-я→-ята）

11. Це учні. — 这些是学生。（阳性：учень→учні，-ь→-і）

12. Очі там. — 眼睛在那里。（不规则：око→очі，к→ч）

13. Це подарунки. — 这些是礼物。（阳性：подарунок→подарунки，飘忽元音）

14. Парасольки тут. — 雨伞们在这里。（阴性：парасолька→парасольки，-а→-и）

15. Це пера. — 这些是羽毛。（中性：перо→пера，-о→-а）

16. Люди там. — 人们在那里。（完全不规则：людина→люди）

17. Це курчата. — 这些是小鸡们。（幼崽：курча→курчата，-а→-ата）

18. Лампи тут. — 灯们在这里。（阴性：лампа→лампи，-а→-и）

19. Це імена. — 这些是名字。（不规则：ім'я→імена，-'я→-ена）

20. Гриби там. — 蘑菇在那里。（阳性：гриб→гриби，辅音+и）

4. 本周小结

第2周完成！

恭喜你完成了第2周「名词性别巩固 + 名词复数」的全部学习！本周你掌握了以下核心技能：

性别判断（第1周 + 本周强化）

  1. 辅音结尾 → 阳性（стіл, брат, вітер）
  2. -а/-я 结尾 → 阴性（школа, сестра, квітка）
  3. -о/-е 结尾 → 中性（вікно, літо, море）
  4. -ння/-ття → 中性，-ість → 阴性，-ь → 记忆
  5. 指男性 → 阳性，外来词 → 中性

复数变化（本周新学）

  1. 阳性：辅音+и，-ь→і，ж/ч/ш/щ+і，飘忽元音脱落
  2. 阴性：-а→и，-я→і，-ь→і，-ість→ості
  3. 中性：-о→а，-е→я，-ння/-ття不变，外来词不变
  4. 不规则：дитина→діти，людина→люди，око→очі
  5. 幼崽名词：-а→-ата / -я→-ята（курча→курчата，теля→телята）
  6. -м'я 类：-'я→-ена（ім'я→імена）
