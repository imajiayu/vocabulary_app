第2周 第3天 — 阳性名词复数 (-и/-і)

学习目标：掌握阳性名词的复数变化规则，学会把单数名词变成复数形式。理解"飘忽元音"现象。

1. 快速回顾

昨天我们强化了性别判断。今天开始学复数（множина）！在乌克兰语中，名词从单数变复数时，词尾会改变。不同性别的名词有不同的复数规则。今天只学阳性名词的复数。

为什么先学阳性复数？
因为阳性名词数量最多，而且复数规则相对统一：基本就是加 -и 或 -і。掌握了阳性，阴性和中性就更容易理解。

2. 语法讲解

什么是复数？

跟中文不同，乌克兰语的名词要区分单数（однина）和复数（множина）。一个东西用单数，两个及以上用复数。复数形式通过改变词尾来实现。

规则一：硬辅音结尾 → 加 -и

大多数以硬辅音（б, в, г, д, к, м, н, п, р, с, т, ф, х）结尾的阳性名词，复数加 -и：

单数复数中文
паркпарки公园 → 公园们
автобусавтобуси公交车 → 公交车们
телефонтелефони电话 → 电话们
урокуроки课 → 课们
студентстуденти学生 → 学生们
братбрати兄弟 → 兄弟们
记忆法：硬辅音 + 硬元音 -и，像给名词穿上"硬壳"变成复数。

规则二：嘶擦音 ж, ч, ш, щ 结尾 → 加 -і

以嘶擦音 ж, ч, ш, щ 结尾的阳性名词，复数加 -і：

单数复数中文
ніжножі刀 → 刀们（注意 і→о！）
м'ячм'ячі球 → 球们
віршвірші诗 → 诗们
ключключі钥匙 → 钥匙们

规则三：-ь 结尾 → 去掉 -ь，加 -і

以 -ь 结尾的阳性名词，去掉 -ь，加 -і：

单数复数中文
вчительвчителі老师 → 老师们
гістьгості客人 → 客人们
місяцьмісяці月 → 月们
ученьучні学生 → 学生们
注意 учень → учні：去掉 -ь 后，е 也脱落了！这就是下面要讲的"飘忽元音"。

规则四：飘忽元音 — 变复数时元音消失或变化

这是阳性复数最大的难点！有些名词变复数时，词干中的元音会消失或变化：

类型A：і → о（最常见）

单数复数中文变化
стілстоли桌子стіл → столи
дімдоми家дім → доми
рікроки年рік → роки
ніжножі刀ніж → ножі

类型B：元音脱落

单数复数中文变化
деньдні天день → дні（е 脱落）
ученьучні学生учень → учні（е 脱落）

类型C：-ок/-ець 的飘忽元音

单数复数中文变化
будинокбудинки楼房будинок → будинки（о 脱落）
подарунокподарунки礼物подарунок → подарунки（о 脱落）
кутоккутки角落куток → кутки（о 脱落）
олівецьолівці铅笔олівець → олівці（е 脱落）
中文母语者易错点：
中文没有单复数变化，所以很容易忘记变形。记住：乌克兰语中说"两张桌子"不是"два стіл"，而是"два столи"！名词必须变复数。

规则五：辅音交替 г → з

少数以 -г 结尾的名词，变复数时 г 变成 з：

单数复数中文变化
другдрузі朋友друг → друзі（г→з）

注意：这是特殊情况，大多数 -г 结尾直接加 -и，如 сніг → сніги。друг → друзі 需要单独记忆。

阳性复数规则总结

词尾复数变化例子
硬辅音 (б,в,д,к,м,н,п,р,с,т...)加 -ипарк → парки
嘶擦音 (ж,ч,ш,щ)加 -іключ → ключі
-ь去ь + -івчитель → вчителі
-око脱落 + -кибудинок → будинки
-ецье脱落 + -ціолівець → олівці

飘忽元音口诀：стіл→столи, дім→доми, рік→роки — "і 变 о，别忘掉！"

3. 例句

通过例句感受单数→复数的转变：

规则复数

1. Це парк. → Це парки. — 这是公园。→ 这些是公园。

2. Студент тут. → Студенти тут. — 学生在这里。→ 学生们在这里。

3. Це підручник. → Це підручники. — 这是教科书。→ 这些是教科书。

4. Автобус там. → Автобуси там. — 公交车在那里。→ 公交车们在那里。

5. Гриб тут. → Гриби тут. — 蘑菇在这里。→ 蘑菇们在这里。

嘶擦音复数 (+і)

6. Це ключ. → Це ключі. — 这是钥匙。→ 这些是钥匙。

7. М'яч тут. → М'ячі тут. — 球在这里。→ 球们在这里。

8. Це вірш. → Це вірші. — 这是诗。→ 这些是诗。

-ь 结尾复数 (去ь+і)

9. Вчитель тут. → Вчителі тут. — 老师在这里。→ 老师们在这里。

10. Це гість. → Це гості. — 这是客人。→ 这些是客人。

飘忽元音复数

11. Стіл тут. → Столи тут. — 桌子在这里。→ 桌子们在这里。（і→о）

12. Це дім. → Це доми. — 这是家。→ 这些是家。（і→о）

13. Рік — це рік. → Роки — це роки. — 年就是年。→ 年们就是年们。（і→о）

14. Ніж там. → Ножі там. — 刀在那里。→ 刀们在那里。（і→о, +і）

15. День тут. → Дні тут. — 天在这里。→ 天们在这里。（е脱落）

-ок/-ець 复数

16. Це будинок. → Це будинки. — 这是楼房。→ 这些是楼房。（о脱落）

17. Подарунок тут. → Подарунки тут. — 礼物在这里。→ 礼物们在这里。（о脱落）

18. Олівець там. → Олівці там. — 铅笔在那里。→ 铅笔们在那里。（е脱落）

特殊复数

19. Це друг. → Це друзі. — 这是朋友。→ 这些是朋友。（г→з+і 特殊！）

20. Учень тут. → Учні тут. — 学生在这里。→ 学生们在这里。（е脱落）

5. 今日小结

今天的收获

  1. 阳性复数基本规则：硬辅音 +и，嘶擦音(ж/ч/ш/щ) +і，-ь 去掉加 -і
  2. 飘忽元音：стіл→столи、дім→доми、рік→роки（і→о）；день→дні（е脱落）
  3. -ок/-ець 飘忽元音：будинок→будинки、олівець→олівці（元音脱落）
  4. 辅音交替：друг→друзі（г→з，特殊记忆）
  5. 学了 18 个新阳性名词及其复数形式
