第1周 第5天 — 软音符号(ь)结尾的名词

学习目标：掌握以软音符号(ь)结尾的名词的性别判断方法，了解阳性和阴性 -ь 结尾名词的常见模式，记住 -ість 结尾总是阴性的核心规则。

回顾

前三天我们学习了名词性别的基本判断规则，来快速回顾一下：

第1天 — 阳性（чоловічий рід）：
  以辅音结尾 → стіл, парк, автобус, телефон

第2天 — 阴性（жіночий рід）：
  以 -а 结尾 → школа, робота, машина
  以 -я 结尾（单辅音+-я）→ кухня, пісня, надія

第3天 — 中性（середній рід）：
  以 -о 结尾 → вікно, слово, озеро
  以 -е 结尾 → море, сонце, місце
  以 -ння/-ття 结尾 → завдання, життя（双辅音+-я → 中性）

到目前为止，我们可以根据词尾判断大多数名词的性别。但有一种情况我们还没处理——当名词以软音符号 ь 结尾时，它可能是阳性，也可能是阴性！这就是今天要攻克的难点。

为什么难？因为 -ь 不像 -а、-о、-е 那样能直接指向某个性别。以 -ь 结尾的名词，阳性和阴性都有，必须学会区分模式并记忆常见词。

1. 语法讲解

1.1 什么是软音符号 ь？

ь（软音符号）本身不发音，它的作用是让前面的辅音"变软"（即腭化）。比如：

  • день（白天）——末尾的 н 被软化，发音更柔和
  • сіль（盐）——末尾的 л 被软化
  • камінь（石头）——末尾的 н 被软化

关键问题是：以 -ь 结尾的名词，怎么判断性别？

1.2 -ь 结尾名词的性别分布

以 -ь 结尾的名词性别判断：

┌──────────┬─────────────────────────────────────────────────┐
│  阳性     │  指人的男性角色、时间概念、自然界具体事物          │
│  він     │  вчитель, гість, день, місяць                │
├──────────┼─────────────────────────────────────────────────┤
│  阴性     │  以 -ість 结尾的词、抽象情感/品质、部分自然现象    │
│  вона    │  радість, любов, ніч, осінь                  │
└──────────┴─────────────────────────────────────────────────┘
核心难点：-ь 结尾名词没有像 -а→阴性、-о→中性 那样的一刀切规则。你需要掌握一些模式和常见词。好消息是，有一条100%可靠的规则可以帮你搞定很大一部分！

1.3 黄金规则：-ість 结尾 → 总是阴性！

这是今天最重要的一条规则，务必牢记：

★ 黄金规则 ★

以 -ість 结尾的名词 → 100% 阴性（没有例外！）

радість（喜悦）     → 阴性
можливість（可能性） → 阴性
новість（新闻）     → 阴性
швидкість（速度）   → 阴性
ніжність（温柔）    → 阴性
молодість（青春）   → 阴性
类比记忆：-ість 相当于英语的 -ness / -ity 后缀。比如：
possible → possibility（可能性）≈ можливість
young → youth/youthfulness ≈ молодість（青春）
quick → quickness ≈ швидкість（速度）
这些抽象品质名词在乌克兰语中全部是阴性。
为什么这条规则这么有用？因为 -ість 结尾的名词数量极多！乌克兰语中大量表示品质、特征、状态的抽象名词都用 -ість 构成。记住这一条，你就能正确判断数百个词的性别。

1.4 阴性 -ь 结尾的其他常见词

除了 -ість 之外，还有一些常见的阴性 -ь 结尾名词。它们没有统一的词尾模式，需要逐个记忆：

阴性 -ь 结尾名词（非 -ість 类）：

自然/时间类：
  ніч（夜晚）    осінь（秋天）    тінь（影子）

情感/抽象类：
  любов（爱）     кров（血）      смерть（死亡）

已学过的：
  відповідь（回答）  сіль（盐）      подорож（旅行）
助记口诀——"夜、秋、影、爱、血、死"都是阴性：
想象一个场景：在一个秋天的夜晚，影子中有爱与血和死亡的故事……这些充满情感和自然意象的词，都是阴性。
中文口诀记："夜秋影，爱血死"——六个阴性 -ь 结尾词。

1.5 以 -ов 结尾的阴性名词

注意 любов（爱）和 кров（血）这两个词，它们以 -ов 结尾。这类词很少，但都是阴性。记住这两个就够了。

1.6 阳性 -ь 结尾名词的常见模式

接下来看阳性。以 -ь 结尾的阳性名词有几个比较明显的类别：

阳性 -ь 结尾名词常见类别：

① 以 -тель 结尾 → 表示"做某事的人"：
  вчитель（老师）— 来自"教"

② 指男性的人：
  гість（客人）— 默认指男性客人

③ 时间/天体：
  день（白天/天）
  місяць（月份/月亮）

④ 自然界具体事物：
  камінь（石头）
  вогонь（火）
-тель 结尾 → 阳性！
这是另一条非常实用的规则。-тель 结尾的词通常表示"做某事的人"，类似英语的 -er/-or。
вчитель（老师）= 教的人
这类词全部是阳性。

1.7 -ь 结尾名词的性别判断流程

总结一个实用的判断流程：

-ь 结尾名词性别判断流程：

第一步：以 -ість 结尾？ → 阴性（100%可靠）
   радість, можливість, новість...

第二步：以 -тель 结尾？ → 阳性（指人的男性角色）
   вчитель...

第三步：都不是？ → 需要记忆常见词！

常见阳性：день, гість, місяць, камінь, вогонь
常见阴性：ніч, сіль, осінь, тінь, любов, кров, смерть, відповідь, подорож

1.8 阳性 vs 阴性 -ь 结尾对比

把阳性和阴性的 -ь 结尾名词放在一起对比：

阳性 -ь（він）阴性 -ь（вона）
день（白天）ніч（夜晚）
вчитель（老师）любов（爱）
гість（客人）осінь（秋天）
місяць（月/月亮）тінь（影子）
камінь（石头）сіль（盐）
вогонь（火）кров（血）
смерть（死亡）
радість（喜悦）
можливість（可能性）
观察：阴性 -ь 结尾名词的数量比阳性多得多。特别是加上所有 -ість 结尾的词，阴性阵营非常庞大。所以如果你拿不准一个 -ь 结尾名词的性别，猜阴性的正确概率更高——但最好还是记住具体的词！

1.9 有趣的对比：день vs ніч

день（白天）是阳性，ніч（夜晚）是阴性。这对"白天和黑夜"恰好一阳一阴。

联想记忆：
день（白天）→ 阳光 → 阳性 → він
ніч（夜晚）→ 月光（柔和）→ 阴性 → вона
"阳光的白天，阴柔的夜晚"——记住这对词。

加上今天学的内容，我们的性别判断规则表更完整了：

2. 例句

以下例句展示 -ь 结尾名词在句子中的使用。只使用 Це（这是）、тут（在这里）、там（在那里）、а（而/但是）等已学结构。

阳性 -ь 结尾名词例句

1. Це день. — 这是白天。

2. Вчитель тут. — 老师在这里。

3. Гість там. — 客人在那里。

4. Це місяць. — 这是月亮。

5. Камінь тут. — 石头在这里。

6. Вогонь там. — 火在那里。

阴性 -ь 结尾名词例句

7. Це ніч. — 这是夜晚。

8. Це любов. — 这是爱。

9. Осінь тут. — 秋天来了（秋天在这里）。

10. Тінь там. — 影子在那里。

11. Це радість. — 这是喜悦。

12. Це можливість. — 这是可能性（机会）。

13. Це новість. — 这是新闻。

14. Смерть там. — 死亡在那里。

对比例句

15. День тут, а ніч там. — 白天在这里，夜晚在那里。（阳性 vs 阴性）

16. Вчитель тут, а школа там. — 老师在这里，学校在那里。（阳性-ь vs 阴性-а）

17. Камінь тут, а дерево там. — 石头在这里，树在那里。（阳性-ь vs 中性-о）

18. Місяць там, а сонце тут. — 月亮在那里，太阳在这里。（阳性-ь vs 中性-е）

19. Степ там, а море тут. — 草原在那里，海在这里。（阳性辅音 vs 中性-е）

20. Радість — це любов. — 喜悦就是爱。（两个阴性）

4. 今日小结

  1. -ь 结尾名词可以是阳性或阴性，不能仅凭词尾一刀切判断，需要记忆或借助模式。
  2. 黄金规则：-ість 结尾 → 100% 阴性（如 радість、можливість、новість）。这条规则没有例外，可以放心使用。
  3. -тель 结尾 → 阳性（如 вчитель），表示"做某事的人"。
  4. 常见阳性 -ь 结尾词：день（白天）、гість（客人）、місяць（月）、камінь（石头）、вогонь（火）。
  5. 常见阴性 -ь 结尾词：ніч（夜）、осінь（秋天）、тінь（影子）、любов（爱）、кров（血）、смерть（死亡）。
  6. 助记口诀："夜秋影，爱血死"——六个阴性常见词。
  7. день vs ніч 对比：白天=阳性，夜晚=阴性。"阳光的白天，阴柔的夜晚"。
  8. 本课新学 18 个词汇（6 阳性 + 12 阴性）。
  9. 明天将学习常见例外与特殊名词（如 тато、батько 等阳性特例）。
