第2周 第4天 — 名词复数：阴性名词复数 (-и/-і)

学习目标：掌握阴性名词的复数变化规则（-а→-и、-я→-і、-ь→-і），学会17个新阴性名词及其复数形式，并能对比阳性与阴性的复数规则。

1. 回顾：阳性名词复数

上节课要点回顾

阳性名词（辅音结尾）变复数的核心规则：

  • 硬辅音结尾 → 加 -и：брат → брати，студент → студенти，будинок → будинки
  • 软辅音/ж/ч/ш/щ/ц 结尾 → 加 -і：олівець → олівці，ключ → ключі，лікар → лікарі
  • 特殊变化：друг → друзі（г→з），учень → учні（去 -ь 加 -і）
快速回忆口诀：阳性复数看结尾 — 硬辅加 -и，软辅/嘶嘶音加 -і。

2. 语法讲解：阴性名词复数规则

今天我们学习阴性名词如何变成复数。阴性名词的复数变化比阳性更加规则，核心思路是"去掉词尾，加上新词尾"。

规则1：-а 结尾 → 去掉 -а，加 -и

这是最常见的阴性复数规则。大量以 -а 结尾的阴性名词都遵循此规则。

单数复数中文
школашколи学校
книгакниги书
машинамашини汽车
роботароботи工作
дорогадороги路

记忆技巧：就像中文里"一所学校 → 很多学校"，乌克兰语只是把 -а 换成 -и，非常简单！

包含 -ка 的词 → -ки：

单数复数中文
квіткаквітки花
зірказірки星星
ложкаложки勺子

包含 -га 的词 → -ги：

单数复数中文
ноганоги脚/腿
подругаподруги女性朋友
注意：不管 -а 前面是什么辅音（к、г、н、т……），规则都一样 — 去 -а 加 -и。-ка→-ки、-га→-ги、-на→-ни，规律完全一致。

规则2：-я 结尾 → 去掉 -я，加 -і

以 -я 结尾的阴性名词，复数时去掉 -я，加上 -і。注意这里加的是 -і（不是 -и），因为 -я 前面通常是软辅音。

单数复数中文
кухнякухні厨房
пісняпісні歌曲
сім'ясім'ї家庭
мріямрії梦想

-ця 结尾 → -ці：

单数复数中文
вулицявулиці街道
ученицяучениці女学生

-ія 结尾 → -ії：

单数复数中文
станціястанції车站
історіяісторії历史
професіяпрофесії职业
对比记忆：
  • -а 结尾 → 加 -и（硬元音）：школа → школи
  • -я 结尾 → 加 -і（软元音）：кухня → кухні

口诀：硬-а配-и，软-я配-і。和阳性复数的逻辑一样（硬辅音加-и，软辅音加-і），乌克兰语的"硬软配对"非常一致！

规则3：-ь 结尾的阴性名词 → 去掉 -ь，加 -і

以 -ь 结尾的阴性名词，变复数时去掉 -ь，加 -і。

单数复数中文备注
тіньтіні影子规则变化
радістьрадості喜悦-ість → -ості
новістьновості新闻-ість → -ості
можливістьможливості可能性-ість → -ості
швидкістьшвидкості速度-ість → -ості

-ість → -ості 是一个非常规律的模式：只要看到 -ість 结尾的阴性名词，复数一定是 -ості。

注意元音交替现象！

少数以 -ь 结尾的阴性名词在变复数时会发生元音交替（і→о 或 і→е）：

  • ніч → ночі（夜晚：і→о！）
  • сіль → солі（盐：і→о！）
  • осінь → осені（秋天：і→е！）

这些需要特别记忆，好在数量不多。

阴性复数规则总结

词尾变化方式示例
-а去 -а 加 -ишкола → школи
-ка→ -киквітка → квітки
-га→ -гинога → ноги
-я去 -я 加 -ікухня → кухні
-ця→ -цівулиця → вулиці
-ія→ -іїстанція → станції
-ь (阴性)去 -ь 加 -ірадість → радості

阳性 vs 阴性复数对比记忆：
性别变化示例
阳性（硬辅音）加 -ибрат → брати
阳性（软辅音）加 -іолівець → олівці
阴性（-а）去-а 加-ишкола → школи
阴性（-я）去-я 加-ікухня → кухні

规律：硬音配 -и，软音配 -і，不管阳性还是阴性！

3. 例句

以下例句展示阴性名词的单数和复数用法。句式使用 Це（这是）、тут（在这里）、там（在那里）。

1. Це тарілка. — 这是盘子。（单数）

2. Це тарілки. — 这些是盘子。（复数：-ка→-ки）

3. Ложка тут. — 勺子在这里。（单数）

4. Ложки там. — 勺子们在那里。（复数：-ка→-ки）

5. Це виделка. — 这是叉子。（单数）

6. Виделки тут. — 叉子们在这里。（复数：-ка→-ки）

7. Це чашки. — 这些是杯子。（复数：-ка→-ки）

8. Мама тут. — 妈妈在这里。（单数）

9. Мами там. — 妈妈们在那里。（复数：-а→-и）

10. Це бабуся. — 这是奶奶。（单数）

11. Бабусі тут. — 奶奶们在这里。（复数：-я→-і）

12. Донька там. — 女儿在那里。（单数）

13. Доньки тут. — 女儿们在这里。（复数：-ка→-ки）

14. Це подруги. — 这些是女性朋友。（复数：-га→-ги）

15. Учениці тут. — 女学生们在这里。（复数：-ця→-ці）

16. Лампи там. — 灯在那里。（复数：-а→-и）

17. Це стіни. — 这些是墙。（复数：-а→-и）

18. Кімнати тут. — 房间在这里。（复数：-а→-и）

19. Ручки там. — 笔在那里。（复数：-ка→-ки）

20. Це парасольки. — 这些是伞。（复数：-ка→-ки）

21. Іграшки тут. — 玩具们在这里。（复数：-ка→-ки）

22. Сукні там. — 连衣裙在那里。（复数：-я→-і）

23. Це хустки. — 这些是手帕。（复数：-ка→-ки）

24. Школи там. — 学校在那里。（复数：-а→-и，已学词汇）

5. 今日小结

阴性名词复数 — 三大规则

  1. -а 结尾 → 去-а 加-и：школа→школи，тарілка→тарілки，мама→мами
  2. -я 结尾 → 去-я 加-і：кухня→кухні，сукня→сукні，бабуся→бабусі
  3. -ь 结尾（阴性） → 去-ь 加-і：радість→радості，тінь→тіні

核心口诀：硬-а配-и，软-я配-і，-ь也配-і。

特别注意：少数 -ь 结尾词有元音交替 — ніч→ночі，сіль→солі。
