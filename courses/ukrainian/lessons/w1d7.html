第1周 第7天 — 复习日：名词性别综合练习

学习目标：巩固本周所学的名词性别判断规则，通过大量综合练习加深记忆。今天不引入新语法，专注于复习和实战。

1. 本周规则总复习

规则一：辅音结尾 → 阳性（第2天）

以辅音字母结尾的名词，绝大多数是阳性。代词用 він。

  • стіл（桌子）、парк（公园）、хліб（面包）、телефон（电话）、автобус（公交车）

规则二：-а / -я 结尾 → 通常阴性（第3天）

以 -а 或 -я 结尾的名词，通常是阴性。代词用 вона。

  • -а 结尾：школа（学校）、робота（工作）、машина（汽车）、дорога（路）
  • -я 结尾：кухня（厨房）、пісня（歌曲）、мова（语言）、надія（希望）
  • 常见模式：-ка（зірка）、-ня（пісня）、-ія（історія）、-ця（вулиця）

规则三：-о / -е 结尾 → 通常中性（第4天）

以 -о 或 -е 结尾的名词，通常是中性。代词用 воно。

  • -о 结尾：вікно（窗户）、слово（单词）、молоко（牛奶）、село（村庄）
  • -е 结尾：море（海）、поле（田野）、серце（心脏）、сонце（太阳）

规则四：-ння / -ття 结尾 → 中性（第4天）

以 -ння 或 -ття 结尾的名词是中性，虽然词尾看起来像 -я。

  • -ння：завдання（作业）、питання（问题）、навчання（学习）、читання（阅读）、знання（知识）
  • -ття：життя（生命）、взуття（鞋子）、почуття（感觉）

规则五：-ь 结尾 → 可阳可阴，需要记忆（第5天）

以 -ь 结尾的名词可能是阳性，也可能是阴性，没有绝对规律，需逐词记忆。

  • 阳性 -ь：день（天）、вчитель（老师）、гість（客人）、місяць（月）、камінь（石头）、вогонь（火）
  • 阴性 -ь：ніч（夜）、сіль（盐）、осінь（秋天）、тінь（影子）、любов（爱）、кров（血）

规律：以 -ість 结尾的名词总是阴性：радість（喜悦）、можливість（可能性）、новість（新闻）、швидкість（速度）

规则六：指男性的 -о/-а/-я 结尾 → 阳性（第6天）

当名词指代男性时，即使词尾是 -о/-а/-я，仍然是阳性。自然性别优先于词尾规则。

  • -о 结尾阳性：тато（爸爸）、батько（父亲）、дідо（爷爷）、дядько（叔叔）
  • -а/-я 结尾阳性：суддя（法官）、воєвода（军事长官）、старшина（士官长）、староста（村长）
  • 男性名字：Микола（尼古拉）、Ілля（伊利亚）

规则七：不变化的外来词 → 通常中性（第6天）

从外语借来的不变化名词，通常是中性。

  • кафе（咖啡馆）、метро（地铁）、таксі（出租车）、радіо（广播）、кіно（电影）、меню（菜单）、журі（评委团）

规则八：代词对应关系

性别代词例子
阳性вінстіл — він тут
阴性вонашкола — вона тут
中性воновікно — воно тут
快速判断口诀：
辅音 → 阳，-а/-я → 阴，-о/-е → 中
-ння/-ття → 中，-ість → 阴，-ь → 要记忆
指男性 → 阳，外来词 → 中
він = 阳，вона = 阴，воно = 中
本周易错点汇总：
  1. тато（爸爸）以 -о 结尾 → 不是中性！是阳性（指男性）
  2. завдання（作业）以 -я 结尾 → 不是阴性！是中性（-ння 模式）
  3. ніч（夜）以 -ь 结尾 → 不是阳性！是阴性（需记忆）
  4. любов（爱）以辅音 -в 结尾 → 不是阳性！是阴性（特殊词）
  5. ім'я（名字）以 -я 结尾 → 不是阴性！是中性（-м'я 类特殊词）

3. 本周小结

第1周完成！

恭喜你完成了第1周「名词性别」的全部学习！本周你掌握了以下核心技能：

  1. 辅音结尾 → 阳性（стіл, парк, телефон）
  2. -а/-я 结尾 → 阴性（школа, кухня, пісня）
  3. -о/-е 结尾 → 中性（вікно, море, серце）
  4. -ння/-ття 结尾 → 中性（завдання, життя）
  5. -ь 结尾 → 需记忆，但 -ість → 阴性（радість, можливість）
  6. 指男性 → 阳性，不管词尾（тато, батько, суддя）
  7. 不变化外来词 → 中性（кафе, метро, таксі）
  8. 代词对应：він = 阳性，вона = 阴性，воно = 中性
