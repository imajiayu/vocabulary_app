# 乌克兰语12周语法入门课程大纲

## 课程总览

本课程为中文母语者设计，从零语法基础开始，系统学习乌克兰语核心语法。每周7天：1天词汇预载 + 5天新课 + 1天复习。

---

## 第1周：名词性别（Рід іменників）

**目标**：掌握乌克兰语名词的三种语法性别判断规则

| 天 | 内容 |
|----|------|
| 1 | **词汇预载**：本周全部单词一键添加到背单词App |
| 2 | 三种性别概述：阳性(чоловічий рід)、阴性(жіночий рід)、中性(середній рід)；辅音结尾→阳性规则 |
| 3 | 阴性名词特征：-а/-я 结尾规则；与阳性对比练习 |
| 4 | 中性名词特征：-о/-е/-я 结尾规则；三种性别综合辨析 |
| 5 | 软音符号(ь)结尾的名词：可能是阳性或阴性，需记忆 |
| 6 | 常见例外与特殊名词（如 батько, тато 等） |
| 7 | **复习日**：第1周内容综合练习 |
